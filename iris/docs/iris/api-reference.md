# IRIS API Reference

## Asset Type Endpoints

### List Asset Types
```
GET /unity-catalog/asset-types
```

### Create Asset Type
```
POST /unity-catalog/asset-types
```

### Get Asset Type
```
GET /unity-catalog/asset-types/{full_name}
```

### Delete Asset Type
```
DELETE /unity-catalog/asset-types/{full_name}
```

## Asset Endpoints

### List Assets
```
GET /unity-catalog/assets
```

### Create Asset
```
POST /unity-catalog/assets
```

### Get Asset
```
GET /unity-catalog/assets/{full_name}
```

### Delete Asset
```
DELETE /unity-catalog/assets/{full_name}
```

### List Child Assets
```
GET /unity-catalog/assets/{full_name}/children
```
